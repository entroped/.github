## Get Started

Along with the new technologies and solutions, new problems are arising, constantly increasing the need of open source development. With joining to the open community, our plan to make sure that everyone can access to the latest best practices in frameworks, generative AI integration and cloud development.

The initial plan is to share the resources with the world making anyone capable of doing incredible things.
